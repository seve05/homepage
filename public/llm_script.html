ask-llm-script


Highlight text, screenshot images and prompt your local language model

Some bash/python scripts to integrate use of large language models into the OS and have a way to quickly search text without needing to copy and paste. This comes in handy when looking up information or even coding in new languages where syntax or methods are unknown.

Since Gemma3 is multi-modal I also implemented image recognition in a single python file that gets initiated by a bash script and is best used by binding to a shortcut. This works by taking the data in the buffer after taking a screenshot or copying an image. Turning this from png to binary -> base64 which is the standardized LLM image input format and then into the model.
