talk-to-calendar

Slowly integrating local LLMs into personal ecosystem and finding use-cases. JSON file constitures the calendar data, the LLM makes changes in the output and wraps those in specific tags which get parsed in python. Prototype for a calendar. This could be a use-case because calendars usually operate in the same way thus the models should be able to place things correctly based on probability and training data, most of the time. Running Gemma3:12b relatively fast, only shows up delayed because of the screen-capture. Likely won't get continued because I don't really use a calendar all that much.
